# YOUth-Safety-Insights-Engine-L-T-Initiative

## Project Overview

An end-to-end analytics project analysing rider safety behaviour, helmet compliance and risk patterns using Python, SQL and Power BI.

## Tools Used

- Python (Pandas)
- MySQL
- Power BI
- Excel/CSV


## Project Workflow

Raw Survey Data
↓
Data Cleaning using Python
↓
SQL Analysis
↓
Power BI Dashboard
↓
Business Recommendations


## Key Analysis

- Helmet compliance analysis
- Rider risk segmentation
- Age-wise risk analysis
- Helmet adoption behaviour
- Purchase decision analysis
